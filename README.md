# 📊 Excel Sales Dashboard

## Overview

This project is an interactive Excel dashboard built to analyze and visualize sales data effectively. It transforms raw data into meaningful insights for better decision-making.

The dashboard focuses on key metrics such as revenue, profit, quantity, and regional performance.

---

## 🚀 Features

* Interactive dashboard using slicers
* Key Performance Indicators (KPIs): Total Sales, Profit, Quantity
* Region-wise and category-wise analysis
* Monthly and yearly trend visualization
* Clean and structured data model
* Pivot tables and charts for fast insights

---

## 🛠️ Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

---

## 🧠 How It Works

1. Raw data is cleaned and structured
2. Pivot tables are created for aggregation
3. Charts are linked to pivot tables
4. Slicers are added for interactivity
5. Dashboard layout is designed for clarity

---

## ⚡ Setup Instructions

1. Download the Excel file
2. Open using Microsoft Excel (2016 or later recommended)
3. Use slicers to explore and filter the dashboard
